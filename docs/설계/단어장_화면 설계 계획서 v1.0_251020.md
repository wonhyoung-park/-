## ✅ 확정된 설계 결정 사항

### 결정 사항 1: 메인 윈도우 구조 - **탭 기반 (확정)**

```
┌──────────────────────────────────┐
│ [단어관리] [플래시카드] [시험] ... │
├──────────────────────────────────┤
│                                   │
│  선택된 탭의 내용 표시            │
│                                   │
└──────────────────────────────────┘
```

**선택 이유**:
- ⭐ 모든 기능이 한 화면에 통합
- ⭐ 탭 전환이 직관적이고 빠름
- ⭐ 학생들에게 익숙한 인터페이스
- ⭐ 학습 흐름 유지에 유리

---

### 결정 사항 2: 플래시카드 표시 방식 - **플립 카드 (확정)**

**앞면 (문제 표시)**
```
┌─────────────────┐
│                 │
│     apple       │  [답 보기]
│                 │
└─────────────────┘
```

**뒷면 (정답 표시) - 클릭 후**
```
┌─────────────────┐
│                 │
│     사과        │  [다음]
│                 │
└─────────────────┘
```

**선택 이유**:
- ⭐ 실제 플래시카드를 사용하는 느낌
- ⭐ 능동적 회상 촉진 (Active Recall)
- ⭐ 카드 뒤집기 애니메이션으로 시각적 피드백
- ⭐ 집중력 향상

**구현 방식**:
- PyQt5 QPropertyAnimation을 사용한 Y축 회전 효과
- 회전 시간: 0.3초
- 앞면/뒷면 QWidget 전환

---

### 결정 사항 3: 단어 목록 테이블 컬럼 - **확정**

**필수 컬럼**:
- 번호
- ⭐ (즐겨찾기)
- 영어 단어
- 한국어 뜻

**추가 컬럼** (포함 확정):
- ✅ 메모
- ✅ 오답률
- ✅ 최종 학습일

**최종 테이블 구조**:
```
┌──────────────────────────────────────────────────────┐
│번호│⭐│영어단어  │한국어뜻│메모      │오답률│최종학습│
├──────────────────────────────────────────────────────┤
│ 1  │⭐│ apple   │ 사과   │과일 종류 │20%  │2025-10-13│
│ 2  │  │ book    │ 책     │         │50%  │2025-10-12│
│ 3  │⭐│computer │컴퓨터  │IT 관련   │8%   │2025-10-13│
└──────────────────────────────────────────────────────┘
```

**선택 이유**:
- 메모: 개인화된 학습 노트 작성
- 오답률: 약점 단어 한눈에 파악
- 최종학습일: 복습 주기 관리

---

### 결정 사항 4: 시험 화면 구조 - **단계별 화면 전환 (확정)**

```
시험 설정 화면 → 시험 진행 화면 → 시험 결과 화면
```

**선택 이유**:
- ⭐ 각 단계가 명확히 구분됨
- ⭐ 시험 진행 중 집중력 향상 (다른 요소 최소화)
- ⭐ 실제 시험 환경과 유사
- ⭐ 단계별 UX 최적화 가능

**화면 흐름**:
1. **시험 설정**: 문항 수, 유형, 시간 선택
2. **시험 진행**: 한 문제씩 표시, 전체 화면 활용
3. **시험 결과**: 점수, 오답 분석, 통계

---

### 결정 사항 5: 통계 차트 종류 - **4개 기본 차트 (확정)**

**포함할 차트**:
1. ✅ **일일 학습량 추이** (선 그래프)
   - X축: 최근 7일
   - Y축: 학습 단어 수
   
2. ✅ **주간 학습 시간** (막대 그래프)
   - X축: 요일 (월~일)
   - Y축: 학습 시간 (분)
   
3. ✅ **숙지도 분포** (파이 차트)
   - 레벨 0~5별 단어 수 비율
   
4. ✅ **오답률 Top 20** (막대 그래프)
   - X축: 오답률 (%)
   - Y축: 단어 (영어)

**구현 라이브러리**: matplotlib

**추가 차트는 Phase 2에서 검토**:
- 학습 세션별 정답률 추이
- 월별 학습 단어 수
- 카테고리별 숙지도

---

### 결정 사항 6: 색상 테마 - **라이트/다크 모드 지원 (확정)**

**라이트 모드 (기본)**:
- 배경: `#FFFFFF` (흰색)
- 강조: `#2196F3` (파란색)
- 텍스트: `#212121` (검정)
- 보조: `#F5F5F5` (연한 회색)

**다크 모드**:
- 배경: `#212121` (어두운 회색)
- 강조: `#00BCD4` (청록색)
- 텍스트: `#FFFFFF` (흰색)
- 보조: `#424242` (중간 회색)

**구현 방식**:
- PyQt5 QSS (Qt Style Sheets) 사용
- `resources/styles/light_theme.qss`
- `resources/styles/dark_theme.qss`
- 설정 화면에서 테마 전환 가능

## 📐 화면 설계 상세 사양

### 1. 최소 윈도우 크기
- 최소 너비: 1000px
- 최소 높이: 700px
- 권장 해상도: 1920×1080

### 2. 폰트
- 기본 폰트: 맑은 고딕 (Windows), 나눔고딕 (대체)
- 기본 크기: 14pt (설정에서 변경 가능: 10~24pt)
- 플래시카드 단어: 32pt (큰 폰트)

### 3. 애니메이션
- 카드 플립: 0.3초 Y축 회전
- 탭 전환: Fade 효과 0.2초
- 버튼 호버: 0.1초 색상 전환

### 4. 접근성
- 모든 버튼에 툴팁 제공
- 키보드 단축키 지원
- 충분한 색상 대비 (WCAG 2.1 AA 기준)
- Tab 키로 포커스 이동 가능
